# 快速部署Devpi-Server

本项目用于以`root`身份在Centos快速部署Devpi-Server

# 上手指南

## clone

```
git clone https://github.com/uncleguanghui/centos-devpi.git
cd centos-devpi
```

进来后可以看到

```
.
├── README.md
├── deploy_centos.sh  # 部署脚本
└── devpi.ini  # supervisor配置

0 directories, 3 files
```

## 部署

```
sh deploy_centos.sh
```
